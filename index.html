My Project

Industrial workers health and safety system based on IOT

The project is based on health monitoring system of the industrial workers. In which we used a web UI called wokwi to sense the temperature and the pulse rate of the workers.The values are sent to the cloud, in which another web UI called node red is been connected to it, in which the predefined health conditions of the workers are programmed.The values are compared predefined conditions and the respective message is sent to the supervisor/ admin of the cloud account.By which the corresponding actions can be taken and save the lives of the workers.

Wokwi Output

The temperature and pulse rate of the workers are displayed.

Cloud Output

The values are also updated in cloud platform when connected.

Nodered Output

The graph shows the pulse rate which is generated in Wokwi.
